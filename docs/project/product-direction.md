# Project Product Direction

## Product Direction

The storefront should feel like a premium flower shop rather than a generic ecommerce template, admin dashboard, or SaaS application.

Every UI decision should prioritize:

- Product discovery
- Beautiful floral imagery
- Clear shopping journeys
- Readable product information
- Trust and professionalism
- Elegant visual hierarchy
- Obvious calls to action

Product imagery should always receive greater visual emphasis than decorative interface elements.

Promotional content should support the shopping experience rather than dominate it.

---

## Typography

For Persian (`fa`) pages:

- Use **BYekan** as the primary UI font.
- Apply BYekan consistently across layouts and components before fallback fonts.
- Preserve proper RTL typography and spacing.

For all other locales:

- Use the project's configured Latin font stack.

Typography changes must remain compatible with both current and future locales.
